# Calculator

A simple yet functional calculator app built with Flutter.
This calculator supports basic arithmetic operations with a clean, 
intuitive user interface


---

# Features

- Basic arithmetic operations
- MVVM Architecture
- Simple Calculator
- Clean Architecture


---
